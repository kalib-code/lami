# Sermon Prep Prompt Templates
*Copy-Paste Prompts for ChatGPT (or Any AI)*

---

## How to Use This Cheat Sheet

1. **Copy** the template
2. **Replace** the [BRACKETED PLACEHOLDERS] with your specifics
3. **Paste** into ChatGPT or your AI tool
4. **Verify** everything before using in your sermon

**Remember:** AI scores only 48% on faith questions. Always check theology, verses, quotes, and facts.

---

## Before You Start: Your Context Block

Paste this at the beginning of every AI session to set context:

```
I'm a [YOUR DENOMINATION] pastor at a [SIZE]-member church in [LOCATION].
My congregation is mostly [DEMOGRAPHICS]. I preach [STYLE] sermons,
typically [LENGTH] minutes. I value [YOUR VALUES - e.g., theological precision,
accessible language]. I'm currently in a sermon series on [SERIES TOPIC].
```

**Example:**
```
I'm a Reformed Baptist pastor at a 200-member church in Quezon City.
My congregation is mostly young professionals and families. I preach expository sermons,
typically 30-35 minutes. I value theological precision but accessible language.
I'm currently in a sermon series on the Gospel of John.
```

## 1. Brainstorming Sermon Angles

### Template:
```
I'm preaching on [PASSAGE] to [AUDIENCE DESCRIPTION].

Give me 5 possible sermon angles or "big ideas" for this passage.
For each angle, provide:
- A one-sentence summary
- Why this angle might connect with my audience
- The main tension or question it addresses

Keep suggestions grounded in the text, not generic spirituality.
```

### Example:
```
I'm preaching on Philippians 4:6-7 to a congregation of young professionals
dealing with career anxiety and financial pressure.

Give me 5 possible sermon angles or "big ideas" for this passage...
```

### Verify:
- [ ] Does each angle actually come from the text?
- [ ] Are there any theological distortions?

## 2. Sermon Illustrations

### Template:
```
I need an illustration about [THEME/CONCEPT] for a sermon on [PASSAGE].

The illustration should:
- Resonate with [AUDIENCE DESCRIPTION]
- Avoid [THINGS TO AVOID - e.g., sports, American cultural references]
- Connect to the point that [YOUR SPECIFIC POINT]

Give me 3 options: one from everyday life, one from relationships,
and one from a real-world scenario my congregation might face.
```

### Example:
```
I need an illustration about God's faithfulness for a sermon on Lamentations 3:22-23.

The illustration should:
- Resonate with Filipino families navigating OFW separations
- Avoid American cultural references
- Connect to the point that God's mercies are new every morning even when circumstances don't change

Give me 3 options...
```

### Verify:
- [ ] Is the illustration factually accurate?
- [ ] Does it actually illustrate your point (not just sound nice)?
- [ ] Would your specific congregation connect with this?

## 3. Greek/Hebrew Word Studies

### Template:
```
Explain the [GREEK/HEBREW] word "[WORD]" used in [PASSAGE].

Include:
- The word's semantic range (different meanings in different contexts)
- How it's used elsewhere in Scripture (give 2-3 examples)
- What the original audience would have understood
- How this informs interpretation of this specific passage

Keep explanation accessible for laypeople but academically grounded.
```

### Example:
```
Explain the Greek word "agape" used in John 3:16.

Include:
- The word's semantic range...
```

### Verify:
- [ ] Cross-check with BDAG, Thayer's, or trusted lexicon
- [ ] Verify the other Scripture references exist
- [ ] Check claims about "original audience understanding"

## 4. Historical & Cultural Context

### Template:
```
What historical and cultural context would help my congregation
better understand [PASSAGE]?

Focus on:
- What the original audience would have known/assumed
- Cultural practices or beliefs relevant to this text
- Historical situation of the author and recipients
- Any archaeological or extra-biblical evidence that illuminates the passage

Provide specific, verifiable details, not general statements.
```

### Example:
```
What historical and cultural context would help my congregation
better understand the Parable of the Good Samaritan (Luke 10:25-37)?

Focus on...
```

### Verify:
- [ ] Cross-check claims with commentaries
- [ ] Verify any "archaeological evidence" mentioned
- [ ] Be skeptical of overly specific claims

## 5. Application Points

### Template:
```
Based on [PASSAGE], which teaches [YOUR INTERPRETATION IN ONE SENTENCE],
give me 3-4 practical applications for [AUDIENCE DESCRIPTION].

Each application should:
- Be specific and actionable (not vague like "trust God more")
- Address a real situation my congregation faces
- Flow naturally from the text's teaching
- Include a concrete "this week" step

Ground each application in the text. Don't add ideas the passage doesn't support.
```

### Example:
```
Based on Colossians 3:23-24, which teaches that all work done for the Lord
has eternal significance, give me 3-4 practical applications for
young professionals feeling stuck in unfulfilling jobs.

Each application should...
```

### Verify:
- [ ] Does each application actually come from this text?
- [ ] Are you adding ideas the passage doesn't support?
- [ ] Would these applications be too generic for any passage?

## 6. Sermon Outline Development

### Template:
```
Help me develop a [NUMBER]-point sermon outline for [PASSAGE].

My big idea is: [YOUR MAIN POINT IN ONE SENTENCE]

Requirements:
- Each point should emerge from the text's structure or argument
- Points should build toward the big idea
- Include a transition sentence between each point
- Suggest where illustrations or applications might fit

I'm NOT looking for you to write the sermon—just the structural skeleton.
```

### Example:
```
Help me develop a 3-point sermon outline for Romans 8:28-30.

My big idea is: God is working all things—even suffering—toward
our ultimate good: being transformed into Christ's image.

Requirements...
```

### Verify:
- [ ] Does this outline follow the passage's actual flow?
- [ ] Are you forcing structure onto the text?
- [ ] Does each point genuinely support the big idea?

## 7. Introduction Hooks

### Template:
```
I'm preaching on [PASSAGE] with the big idea: [YOUR MAIN POINT].

Give me 3 introduction hook options:
1. A thought-provoking question
2. A relatable scenario or "imagine this" opener
3. A counterintuitive statement or tension

Each hook should:
- Immediately engage [AUDIENCE DESCRIPTION]
- Create tension or curiosity that the sermon will resolve
- Lead naturally into the passage

Keep each hook under 50 words.
```

### Example:
```
I'm preaching on Matthew 6:25-34 with the big idea:
Anxiety reveals what we're actually trusting instead of God.

Give me 3 introduction hook options...
```

### Verify:
- [ ] Does the hook actually connect to your point?
- [ ] Is it appropriate for your congregation?
- [ ] Does it create genuine tension the sermon resolves?

## 8. Anticipating Objections

### Template:
```
I'm preaching on [PASSAGE] and making the claim that [YOUR TEACHING].

What are the 3 most likely objections or questions someone in my
congregation might have?

For each objection:
- State the objection clearly
- Explain why someone might think this
- Provide a brief biblical response
- Suggest how I might address this pastorally (not just intellectually)
```

### Example:
```
I'm preaching on 1 Timothy 2:11-12 and making the claim that
Paul is establishing a complementarian pattern for church leadership.

What are the 3 most likely objections or questions...
```

### Verify:
- [ ] Are the objections realistic for your congregation?
- [ ] Are the responses biblically sound?
- [ ] Check any Scripture references in the responses

## 9. Polishing Your Writing

### Template:
```
Here's a paragraph from my sermon. Help me improve it while keeping my voice.

[PASTE YOUR PARAGRAPH]

Options (pick one):
- Make it more concise (cut by 30% without losing meaning)
- Make it more vivid (add sensory language or concrete details)
- Make the flow smoother (improve transitions)
- Make it more accessible (simplify without dumbing down)
- Strengthen the emotional impact

Show me the revised version and briefly explain what you changed.
```

### Example:
```
Here's a paragraph from my sermon. Help me improve it while keeping my voice.

"God's grace is really amazing when you think about it. We don't deserve it
but He gives it anyway. This should make us feel grateful and want to live
differently. When we understand grace, everything changes."

Make it more vivid (add sensory language or concrete details).
```

### Verify:
- [ ] Does it still sound like you?
- [ ] Did the meaning shift at all?
- [ ] Is the theology still accurate?

## 10. Cross-References & Related Passages

### Template:
```
What other Scripture passages speak to the same theme as [PASSAGE]?

I'm focusing on the theme of: [SPECIFIC THEME]

Give me:
- 3-5 Old Testament passages that connect
- 3-5 New Testament passages that connect
- For each, explain briefly how it relates (don't just list verses)

Prioritize passages that would strengthen my sermon's argument,
not just verses that mention similar words.
```

### Example:
```
What other Scripture passages speak to the same theme as 1 Peter 5:7?

I'm focusing on the theme of: casting anxiety on God because He cares for us

Give me...
```

### Verify:
- [ ] Look up EVERY reference—AI invents verses
- [ ] Check if the connection is actually thematic or superficial
- [ ] Read each passage in context

## Quick Reference: What to Always Verify

| AI Output | Verify Against |
|-----------|----------------|
| Bible verses | Your Bible (AI invents references) |
| Greek/Hebrew words | Lexicons (BDAG, Thayer's, BDB) |
| Historical claims | Commentaries & academic sources |
| Quotes | Original source (AI misattributes) |
| Statistics | Primary research |
| Theological claims | Your confessional standards |
